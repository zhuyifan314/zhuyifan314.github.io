0/1 分数规划

2021-07-05 3 min read

一、简介

0/1 分数规划是一种求最优比率的算法，一般通过二分实现。

二、解决问题

分数规划是用来求解一个分数表达式的最大值/最小值。例如有 nnn 个物品，每个物品都有两个权值 aia_iai​ 和 bib_ibi​，你可以取其中任意件物品，求 ∑i=1kai∑j=1kbj\frac{\sum_{i=1}^k a_i}{\sum_{j=1}^k b_j}∑j=1k​bj​∑i=1k​ai​​ 最大值。

三、解决方法

我们一般通过二分来求解。

我们二分最终答案 ansansans，设 val=∑i=1kai,wei=∑j=1kbjval=\sum_{i=1}^k a_i,wei=\sum_{j=1}^k b_jval=∑i=1k​ai​,wei=∑j=1k​bj​，那么显然有

valwei≤ans\frac{val}{wei}\leq ans weival​≤ans

移项得

val≤ans×weival\leq ans\times wei val≤ans×wei

val−ans×wei≤0val-ans\times wei\leq 0 val−ans×wei≤0

∑i=1kai−ans×∑j=1kbj≤0\sum_{i=1}^k a_i-ans\times \sum_{j=1}^k b_j\leq 0 i=1∑k​ai​−ans×j=1∑k​bj​≤0

∑i=1kai−ans×bi≤0\sum_{i=1}^k a_i-ans\times b_i\leq 0 i=1∑k​ai​−ans×bi​≤0

一般我们贪心地选择 kkk 个然后判断是否小于等于 000。
