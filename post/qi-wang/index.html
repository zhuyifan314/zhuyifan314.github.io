一、什么是期望

在 OI 中我们常指的期望是达到结果的期望，简单来说就是 每次可能结果的概率乘以其结果的总和。

一个离散性随机变量 XXX 的数学期望是每个取值乘以该取值对应概率的总和，记为 E(X)E(X)E(X)。

E(X)=∑α∈I(X)α⋅P(X=α)=∑ω∈SX(ω)P(ω)E(X) = \sum_{\alpha\in I(X)} \alpha \cdot P(X=\alpha) = \sum_{\omega\in S}X(\omega)P(\omega) E(X)=α∈I(X)∑​α⋅P(X=α)=ω∈S∑​X(ω)P(ω)

其中 I(X)I(X)I(X) 表示随机变量 XXX 的值域，SSS 表示 XXX 所在概率空间的样本集合。

二、性质

  • 设 XXX 是随机变量，CCC 是常数，则 E(CX)=C×E(X)E(CX) = C \times E(X)E(CX)=C×E(X)。

证明：设 XXX 的多个随机变量为

Ca1,Ca2,Ca1,…,CanC_{a_1},C_{a_2},C_{a_1},\ldots,C_{a_n} Ca1​​,Ca2​​,Ca1​​,…,Can​​

p1,p2,p3,…,pnp_1,p_2,p_3,\ldots,p_n p1​,p2​,p3​,…,pn​

那么对应的求期望的式子

E(CX)=C∑i=1n(ai×pi)E(CX) = C\sum_{i=1}^{n}(a_i\times p_i) E(CX)=Ci=1∑n​(ai​×pi​)

将 CCC 提出，因为

E(X)=∑i=1n(ai×pi)E(X) = \sum_{i=1}^n(a_i\times p_i) E(X)=i=1∑n​(ai​×pi​)

所以

E(CX)=C×E(X)E(CX) = C \times E(X) E(CX)=C×E(X)

  • 设 X,YX,YX,Y 是任意两个随机变量，则有 E(X+Y)=E(X)+E(Y)E(X+Y) = E(X) + E(Y)E(X+Y)=E(X)+E(Y)。

  • 设 X,YX,YX,Y 是任意两个相对独立的随机变量，则有 E(XY)=E(X)×E(Y)E(XY) = E(X) \times E(Y)E(XY)=E(X)×E(Y)。

  • 设 CCC 为常数，则 E(C)=CE(C) = CE(C)=C。

三、赠券收集问题

维基百科

SP1026 FAVDICE - Favorite Dice

其实挺好理解的，相当于期望选 nnn 个球才选到 111 个红球，那么选到红球的概率为 1n\frac{1}{n}n1​。

而根据期望的性质：设 X,YX,YX,Y 是任意两个随机变量，则有 E(X+Y)=E(X)+E(Y)E(X+Y) = E(X) + E(Y)E(X+Y)=E(X)+E(Y)，使得每一面都被掷到的期望就是 ∑i=1nnn−i+1=∑i=1nni\sum_{i=1}^n \frac{n}{n-i+1} = \sum_{i=1}^n \frac{n}{i}∑i=1n​n−i+1n​=∑i=1n​in​。

至于如何预处理，设计算到第 iii 个数，a,ba,ba,b 是 iii 前面分数的前缀和的分子和分母，则有 ab+1i=a×i+bb×i\frac{a}{b}+\frac{1}{i}=\frac{a \times i + b}{b \times i}ba​+i1​=b×ia×i+b​，然后转移即可。
