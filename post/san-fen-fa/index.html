三分法

2021-05-09 1 min read

一、用处

三分法使用于求单峰函数的极值。

二、流程

  1. 先将区间三分，每个区间的长度为 13(r−l)\frac{1}{3} (r - l)31​(r−l)；

  3. 重复上述流程，直到不满足 r−l>epsr - l > epsr−l>eps 为止。

三、代码

	while(r - l > eps) {
		mid1 = l + (r - l) / 3;
		mid2 = r - (r - l) / 3;

		else

	}
